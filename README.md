# 📚 Библиотечная система (консольное приложение)

Простая консольная программа на C++ для управления списком книг в библиотеке.  
Позволяет добавлять, просматривать, редактировать и удалять записи о книгах.

## 🛠 Функциональность

- **Добавление книги**  
  Ввод названия, автора и жанра. Данные сохраняются в трёх параллельных векторах.

- **Просмотр всех книг**  
  Отображает список всех добавленных книг с нумерацией, автором и жанром.

- **Удаление книги**  
  Удаление по названию (удаляются все связанные данные из трёх векторов).

- **Редактирование книги**  
  Изменение названия, автора и жанра для книги с заданным названием.

- **Выход из программы**  
  Завершает цикл работы приложения.

## ⚙️ Технические детали

- Язык: **C++**
- Хранение данных: три вектора `vector<string>`:
  - `vbooks` — названия книг
  - `vauthor` — авторы
  - `vgenres` — жанры
- Локализация: `setlocale(LC_ALL, "Ru")` для поддержки русского языка в консоли
- Защита от некорректного ввода номера действия: `cin.fail()`, `cin.clear()`, `cin.ignore()`
